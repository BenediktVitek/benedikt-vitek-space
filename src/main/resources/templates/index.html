About this site

The main purpose of creating this website is to showcase some of my skills. While you can certainly check my GitHub to see the projects I have been working on or view my CV to see what I claim to know, I also wanted something a bit more interactive to demonstrate that I can deliver. Since I have primarily been learning Java, I have developed some interactive sections that you can explore in the showcase part of this site.

I would like to walk you through these sections. There are five different modules available for interaction, with some interconnected elements. Starting from the left, the first module displays the top ten most-watched movies on Netflix in the US, obtained through web scraping. In the second module, I use a third-party API (The Movie Database) to fetch information about movies identified by the web scraper. This information is utilized to display the ratings of the movies and create links to TMDB.

The third module allows you to register a new account or log in with an account you've registered before. Don't worry; you don't have to create an extremely complicated password, as I didn't want to burden you with this feature. However, it's still advisable to use a strong password. New users are saved in the database, and every password is encoded. I implemented this part using Spring Security.

The fourth module enables you to leave comments on this site, either anonymously or using a created user. This creates a many-to-many relationship between users and their messages. The last module utilizes my created API endpoints for reading and creating. Unfortunately, update and remove endpoints were not implemented in this project, but if you would really miss them, then I can always implement them. If you use one of the buttons in this module, the site will respond with JSON according to your choice.
