About this site

Welcome to my showcase, where I demonstrate my proficiency in Java through interactive sections highlighting various skills. Here's a breakdown of what you'll find:

Section 1 - Web scraping: Explore the top 10 most-watched movies on Netflix obtained through web scraping. Titles are encoded for use in the following section.

Section 2 - Using third party APIs: Discover movie information sourced from TMDB APIs, including ratings and direct links to each movie on TMDB.

Section 3 - Spring security: Experience the functionality of user login and registration. Password strength is intentionally not enforced for easy testing. Registering a user creates a user entity stored in the database.

Section 4 - Relational database: Add comments as an anonymous user or under your registered user profile. This establishes a relationship between user and comment entities, saving the comments in the database.

Section 5 - Building APIs: Explore comments made on this site. Each response is crafted using DTOs, and the response format is in JSON.

Feel free to navigate through these sections to witness firsthand the practical application of my Java skills. Don't hesitate to provide feedback or explore my GitHub and my resume for a comprehensive view of my projects and expertise.
